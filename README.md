# raylib js bunnymark
A quick bunnymark to test performance of the `raylib` npm module.

Thanks to https://github.com/arthurmassanes/flappy for making me interested and providing the basis for this!

## Requirements:
`cmake`

`node (>= 12)`

# How to install & run

```bash
yarn install
yarn start
# spacebar to add bunnies
# r to reset
```
